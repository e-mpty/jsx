# Basic React Setup 

This is my basic react setup for any simple project or learning. 
In this basic setup You have two theme already set  (dark and light), Icons already set.

 ## CSS Framework:


 - [TailwindCSS](https://tailwindcss.com/docs/installation/using-vite) : The basic CSS Framework for all my projects 
 - [DaisyUI](https://daisyui.com/docs/install/) : The main CSS Framework for all mny projects 

 ## Icon FrameWork:

 - [Remix Icon](https://remixicon.com/) : An open source Icons resources.

 ## Fonts:

 - Space Mono
 - Space Grotesk
 - Lato

>All available on [Google Fonts](https://fonts.google.com/)

## How to use it:
> - Pull the repo
> - Run `npm install` and then `npm run dev`
